📝 Guide de configuration Google Sheets (Apps Script)

  1. Ouvrez votre fichier Google Sheet : Voeux classes 2026 LFJP
  2. Dans le menu supérieur, allez dans Extensions > Apps Script.
  3. Supprimez le code existant dans l'éditeur (souvent function myFunction() { }).
  4. Copiez le code ci-dessous et collez-le dans l'éditeur : 
    Chargement du code...
  5. Cliquez sur la disquette 💾 (Enregistrer le projet).
  6. Cliquez sur le bouton Déployer (en haut à droite) > Nouveau déploiement.
  7. Sélectionnez le type Application Web (cliquez sur l'engrenage à côté de "Sélectionner un type").
  8. Configurez les options suivantes :
    • Description : API Constitution des Classes 2026/2027
    • Exécuter en tant que : Moi (votre adresse email)
    • Qui a accès : Tout le monde (Anyone) — Note : Cela est requis pour que l'application cliente puisse envoyer des données sans authentification complexe.
  9. Cliquez sur Déployer. Google vous demandera d'autoriser les accès. Validez-les (cliquez sur "Paramètres avancés" puis sur "Accéder à Projet sans titre (non sécurisé)" puis "Autoriser").
  10. Copiez l'URL de l'application Web générée (elle se termine par /exec) et collez-la dans le champ ci-dessus.
  11. Dans votre Google Sheet, allez dans Fichier > Partager > Publier sur le Web. Sélectionnez la première feuille et publiez-la au format Page web. Copiez cette URL de publication (ou simplement l'URL de votre sheet d'édition) et collez-la dans le premier champ de configuration.
